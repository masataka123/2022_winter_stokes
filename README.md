# 成績に関して
次の1と2を満たしているものに単位を与えます.

1. 幾何学1の講義の単位が可以上である. 
2. 13回授業時までに演習の授業で少なくとも1回以上発表している.

ただし2の条件を達成できないものには別途救済レポートを課して2を達成したものとすることがある.
